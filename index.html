关于汉字点阵字库

            
本编辑器支持 HZK16 和 HZK12 点阵汉字库，用于在 LCD 上显示中文字符。

            
要使用汉字功能，您需要：

            
    
                
  1. 获取 HZK16 和/或 HZK12 字库文件（网上可搜索下载）
    2. 
                
  2. 将字库文件命名为 "HZK16" 和 "HZK12"（无扩展名）
    3. 
                
  3. 放置在网页同级目录下的 fonts 文件夹中
    4. 
            

            
技术说明：

            
    
                
  • HZK16 是 GB2312 编码的 16×16 点阵汉字库，包含 6763 个汉字和 682 个符号。
    • 
                
  • HZK12 是 GB2312 编码的 12×12 点阵汉字库，同样支持 GB2312 标准中的汉字和符号。
    • 
                
  • 每个汉字在 HZK16 中占用 32 字节存储空间，在 HZK12 中占用 24 字节存储空间。
    • 
                
  • 本程序使用 Unicode 字符直接转换为 GB2312 编码进行显示，不再使用预设映射表。
    • 
            

            
调试信息：

            
您可以通过点击"调试信息"按钮查看当前汉字的编码和偏移量信息，便于与 Python 代码进行比较。
